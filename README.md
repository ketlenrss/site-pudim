# 🍮 O Novo Pudim

Uma recriação moderna do clássico site brasileiro `pudim.com.br`, desenvolvida como projeto acadêmico de Engenharia de Software. O objetivo foi modernizar o visual utilizando boas práticas de desenvolvimento web, mantendo a essência e a história do doce.

🔗 **[Acesse o site publicado aqui!](https://novo-pudim.netlify.app/)**

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estrutura e semântica.
* **CSS3:** Estilização, animações, Flexbox e Grid.
* **Netlify:** Hospedagem e deploy.
